Avenir Performance Marketing Academy - No Login + AdSense Ready Version

Created by Maninder Singh

What changed in this version:
- Login/logout system removed completely.
- The academy opens directly from dashboard.html/modules.html.
- login.html and js/auth.js were removed.
- No page redirects visitors to login.
- AdSense placeholder spaces are included across pages.
- adsense-setup.html explains where to paste AdSense scripts and ad unit code.

Deploy on GitHub + Vercel:
1. Upload all files and folders to GitHub repo root.
2. Repo root should directly contain index.html, css/, js/, and all pages.
3. Connect repo to Vercel.
4. After deployment, open your website and press Ctrl + Shift + R.

AdSense setup after approval:
1. Open each HTML page.
2. Paste the AdSense global script inside <head> near the AdSense placeholder comment.
3. Replace <div class="ad-slot"> placeholder content with AdSense ad unit code.
4. Keep Terms, Privacy, About, Contact, and Blog pages live for policy trust.

Important:
Do not remove css/ or js/ folders. The design and app features need them.
